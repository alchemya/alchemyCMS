# alchemyCMS
一款自娱自乐的基于flask的企业级CMS系统
# 项目概述
大概一年多以前为了练习flask而做着玩的，但是完成度极其高。
同时是为了锻炼自己的前端能力，就搞事情自己写了前端界面  
flask+sqlalchemy+jinja2+bootstrap+jquery
表单传输大量使用了我封装的ajax，效果拔群
实现了权限管理和多级用户权限
实现了前后端两套系统逻辑
